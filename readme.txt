== Description ==

AI Block Theme Template is a comprehensive starter template for WordPress block theme development. It provides a complete foundation for creating modern, full-site editing themes with AI-powered development features.

= Features =

* Full Site Editing (FSE) support
* Block patterns and template parts
* Custom color palettes and typography
* Style variations including dark mode
* Responsive design with fluid typography
* Modern development workflow with build tools
* End-to-end testing with Playwright
* AI-powered development assistance
* GitHub Copilot integration
* Comprehensive documentation

= Block Patterns =

The theme includes a variety of block patterns to help you quickly build page layouts:

* Header and footer patterns
* Hero sections
* Content layouts
* Call-to-action sections

= Style Variations =

* Default light theme
* Dark theme variation
* Customizable color schemes

= Development Features =

* Modern build tools (webpack, PostCSS)
* Code linting and formatting (ESLint, Prettier, Stylelint)
* PHP code standards (PHPCS)
* Automated testing
* AI development assistance

== Installation ==

1. Download the theme files
2. Upload to your WordPress site's themes directory
3. Activate the theme from the WordPress admin
4. Customize using the Site Editor

For development:

1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `composer install` for PHP dependencies
4. Use `npm run start` for development
5. Use `npm run build` for production builds

== Frequently Asked Questions ==

= What WordPress version is required? =

WordPress 6.0 or higher is required for full site editing features.

= Does this theme support classic editor? =

This theme is designed specifically for the block editor and full site editing. Classic editor support is limited.

= Can I customize the colors and fonts? =

Yes, the theme provides extensive customization options through the Site Editor and theme.json configuration.

= Is this theme suitable for production sites? =

Yes, this theme is production-ready and includes performance optimizations and security best practices.
