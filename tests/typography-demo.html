Typography Token System Demo

This demo shows how the refactored typography tokens scale fluidly across viewport sizes.

Typography Scale (100-900)

font-size-900 (Colossal) - Hero H1 primary
font-size-800 (Gigantic) - Hero H1 large
font-size-700 (Huge) - H2 / hero sub-heading
font-size-600 (X-Large) - H3
font-size-500 (Large) - H4 / feature headings
font-size-400 (Medium) - H5/H6 / emphasized text
font-size-300 (Small) - Lead paragraph / small subhead
font-size-200 (Base) - Paragraph body
font-size-100 (Tiny) - Fine print, meta micro-labels

  • ✅ Consistent with WCAG 2.2 AA guidelines
  • ✅ Predictable scaling behavior
